# engle.js
Library for handling the last name 'Engle'

```
engle();
// Returns the string 'Engle'.

engle(string);
// Returns true if string is 'Engle'.

engle.reverse();
// Returns 'Engle' backwards.

engle.ascii();
// Returns ascii code for 'Engle'.

engle.binary();
// Returns binary code for 'Engle'.

engle.octal();
// Returns octal code for 'Engle'.

engle.hex();
// Returns hex code for 'Engle'.

engle.list();
// Returns array of Engles.
```
